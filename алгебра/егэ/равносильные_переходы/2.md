**Как избавиться от арифметического корня**

--- ---

$h \cdot \left( \sqrt f - \sqrt {g \vphantom f} \right) \ \vee 0 \ \ \ \ \ \Longleftrightarrow \ \ \ \ \ \begin{cases} h \cdot (f-g) \ \vee \ 0, \\\\ f \geqslant 0, \ g \geqslant 0. \end{cases}$
